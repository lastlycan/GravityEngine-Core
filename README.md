# GravityEngine Core 🚀

The open-source foundation for building autonomous Salesforce QA swarms with Playwright.

## 📦 What's Inside?
- **BasePage**: Standardized Salesforce record interaction patterns.
- **SalesforceApi**: Lightweight REST API wrapper for ultra-fast data setup/cleanup.
- **GlobalNavigationBar**: Intelligent navigation across Sales Cloud and Service Console.
- **Auth Utilities**: Session extraction and MFA-aware global setup templates.

## 🛠️ Installation
```bash
npm install @gravityengine/core
```
